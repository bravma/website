# Dr. Marc Bravin — Personal Website

Static personal website for Dr. Marc Bravin, CTO & Co-Founder at Gopf AG, AI Researcher and Lecturer at HSLU.

## Structure

```
index.html      # About / landing page
work.html       # Work experience
research.html   # Research & talks
projects.html   # Projects
press.html      # Press & awards
contact.html    # Contact
assets/
  style.css     # Shared stylesheet
  foto_marc_bravin.jpeg
```

## Development

No build step — open any `.html` file directly in a browser or serve with any static file server:

```bash
npx serve .
# or
python3 -m http.server
```
